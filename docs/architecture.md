## 数据流

### 用户操作板子
```
用户 → 前端(Vue) → API(/api/boards/{id}/exec) → BoardManager(SSH) → 板子
                                                      ↓
                                                  返回结果 → 前端显示
```

### AI 操作板子
```
用户 → AI聊天页面 → API(/api/experiments/ai-chat/{board_id})
         → AIAgent.chat() → Anthropic API → Tool Use
         → exec_command/create_experiment_plan → BoardManager/DB
         → 返回回复
```

### 预约流程
```
用户 → 选择板子+时间段 → Scheduler.create_booking()
         → 检查冲突 → 锁定板子 → 返回预约ID
         → 到时间自动激活 → 用户开始实验 → 完成释放
```

## 关键技术决策

1. **异步优先**: 使用 FastAPI + async/await + asyncio，支持并发连接多块板子
2. **连接池复用**: BoardManager 维护板子连接池，避免每次操作重新建连
3. **AI 安全三层防护**: 结构化工具(不能自由写命令) → 高危操作禁用在代码层 → 人工确认
4. **JWT 无状态认证**: 适合微服务化扩展，不需要 session 存储
