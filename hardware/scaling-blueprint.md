## 板子连接服务器的硬件方案（重点）

这是搭建机房的核心问题：数十到数百块板子如何物理上放置、连接供电、接入网络/串口到服务器。

### 1. 放置板子的物理方案

| 方案 | 设备 | 适用场景 | 预估价格 |
|------|------|---------|---------|
| **亚克力多层架** | 定制亚克力板 + 铜柱 | 10-50块板子，桌面放置 | ~200-500 |
| **19寸机柜托盘** | 1U/2U 固定托盘 | 20-100块，放机柜内 | ~50-100/个 |
| **板卡插箱** | 3U/6U 板卡插箱(带导轨) | 批量插拔，专业场景 | ~500-2000/个 |
| **洞洞板/挂架** | 金属洞洞板 + 扎带 | 灵活摆放，调试方便 | ~100-300 |

**推荐做法：**
- 小规模（10-30块）：`亚克力多层架` 放桌面，每层放 5-8 块板子
- 中规模（30-100块）：`19寸机柜 + 托盘`，每托盘放 4-6 块板子，按项目分托盘
- 大规模（100+）：`板卡插箱` + `机柜`，类似服务器机柜的插拔式管理

### 2. 板子如何连接到服务器

#### 2.1 Linux 板（SSH 连接）

```
服务器 <--网线--> 交换机 <--网线--> 板子(RPi/Orange Pi/Jetson)
```

| 设备 | 型号示例 | 数量 | 单价(RMB) |
|------|---------|------|----------|
| **千兆交换机** | H3C S5024PV5-EI (24口) | 2-4台 | ~800 |
| **或 千兆交换机** | TP-LINK TL-SG1024DT (24口) | 2-4台 | ~500 |
| **网线** | 六类网线 0.5米/1米 | 板子数量×1 | ~5-10/根 |
| **线槽/理线架** | 1U 水平理线架 | 每交换机配2个 | ~50 |

> ⚠️ 如果板子支持 WiFi（树莓派等），也可以走 WiFi 减少布线，但百块规模建议有线更稳定。

#### 2.2 MCU 板（串口连接 — 最关键的设备）

MCU 板通过串口（UART/USB转串口）连接。这是硬件方案的核心：

```
服务器 <--USB--> USB Hub <--USB--> USB转串口模块 <--串口线--> 板子
                             或
服务器 <--USB--> 串口集线器 <--串口排线--> 多块板子
```

| 设备 | 型号示例 | 每台可接板子数 | 单价(RMB) | 100块板需要 |
|------|---------|--------------|----------|-----------|
| **USB 3.0 Hub** | 优越者 ORICO 7口/10口 (带供电) | 7-10路 | ~100-200 | 10-15台 |
| **USB转串口模块** | CP2102 / CH340G / FT232RL | 1路 | ~10-20 | 板子数量×1 |
| **多路串口集线器** | USB转8口/16口串口卡 | 8-16路 | ~300-800 | 6-13台 |
| **串口服务器** | USR-N510 / MOXA NPort 5150 | 1路(网络转串口) | ~200-500 | 板子数量×1 |
| **多路串口服务器** | USR-N580 (8口) / MOXA 8口 | 8路 | ~1,500-3,000 | 12-13台 |

**两种方案对比：**

| | 方案A: USB Hub + USB转串口（推荐） | 方案B: 串口服务器 |
|---|------|------|
| 连接方式 | USB 直连服务器 | 网线连接交换机 |
| 100块成本 | ~3,000-5,000 | ~15,000-30,000 |
| 延迟 | 低 | 中 |
| 配置复杂度 | 简单(即插即用) | 需要配IP |
| 推荐 | **✅ 推荐** | 板子分布在不同房间时用 |

**方案A 实物连接示例：**
```
服务器
  │
  ├── USB Hub #1 (10口)
  │    ├── USB转串口#1 → 板子A (CH340)
  │    ├── USB转串口#2 → 板子B (CH340)
  │    ├── ...
  │    └── USB转串口#10 → 板子J (CH340)
  │
  ├── USB Hub #2 (10口)
  │    ├── USB转串口#11 → 板子K
  │    └── ...
  │
  └── USB Hub #3 ... (第3层USB扩展)
```

⚠️ **注意事项：** USB Hub 最多级联 3-4 层，总板子数多时服务器后面板 USB 口不够，需用 PCIe USB 扩展卡：

| 设备 | 可增加USB口 | 价格 |
|------|-----------|------|
| PCIe USB 3.0 扩展卡 (4口) | +4 | ~100 |
| PCIe USB 3.0 扩展卡 (7口) | +7 | ~200 |
| 服务器通常可插 2-4 张 | +8~28口 | |

## 运维建议完善版

1. **PXE 网络启动**：批量给 Linux 板装系统，不用插 SD 卡
2. **集中供电**：USB 集线器供电柜，统一控制重启（PDU可远程开关）
3. **串口映射规则**：在服务器上创建 udev 规则，固定 USB 转串口的设备名 `/dev/ttyLAB_01`，避免拔插后名称变化
4. **板子标签**：每块板子贴二维码/编号，扫码可查在系统中的 ID 和位置
5. **分区分组**：按项目/实验类型分 VLAN 隔离
6. **远程复位**：每块板子加一个继电器模块（如 1路 5V继电器），服务器可通过 GPIO/USB 控制板子断电重启
7. **温湿度监控**：机柜内放一个 ESP32 + DHT22 做环境监控，温度过高自动告警
8. **集中日志**：所有板子的串口输出统一发到服务器的 `rsyslog` 或 `logs/` 目录
